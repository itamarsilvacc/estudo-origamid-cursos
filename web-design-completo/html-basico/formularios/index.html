Formulário

Formulários são muito usados na internet, e possibilitam a iteração do usuário com o conteúdo ou sistema onde estão sendo aplicados. Essa iteração, que acontece enviando e recebendo dados de um servidor web, é feita através de uma linguagem de programação, como o PHP.

Abaixo, a codificação de um simples formulário exemplificando o uso de algumas tags relacionadas à formulário HTML.
